
# Project Title

A Simpe COVID tracker built to try out chart.js.





## Demo

https://basilawwad.github.io/covid-tracker/

## Run Locally

Clone the project

```bash
  git clone https://github.com/basilawwad/covid-tracker.git
```

Go to the project directory

```bash
  cd covid-tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
